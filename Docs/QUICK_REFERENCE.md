## Button Styles
- `DynamicResource BaseButton` - Default
- `DynamicResource PrimaryButton` - Accent filled
- `DynamicResource DangerButton` - Red filled
- `DynamicResource SuccessButton` - Green filled
- `DynamicResource GhostButton` - Transparent
