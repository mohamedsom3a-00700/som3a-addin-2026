# Changelog

All notable changes to Som3a Add-in will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.3.0] - 2026-05-28

### Added
- Dashboard & Home page with 9 status widgets (Phase 23)
- Diagnostics dashboard with real-time system metrics
- Recent Tools and Recent Projects tracking
- AI Provider Status monitoring
- Plugin Status overview
- Quick Actions navigation widget

### Changed
- Replaced WelcomePage with Home dashboard as default landing page
- Added Home category to sidebar navigation (order 0)

## [1.2.0] - 2026-05-20

### Added
- Duration Estimator module (Phase 22)
- Relationship Generator module (Phase 21)
- BOQ Activity Generator module (Phase 20)

### Changed
- Enterprise Planning Platform foundation complete (Phases 14-19)

## [1.1.0] - 2026-05-10

### Added
- Shell workspace with sidebar navigation
- Theme engine with Dark/Light/Custom themes
- Command palette (Ctrl+K)
- Settings registry and dynamic settings UI

## [1.0.0] - 2026-04-15

### Added
- Initial release
- Excel VSTO add-in for Primavera P6 integration
- WBS Generator and Editor
- Primavera Compare and XER Editor
- Float Path analysis
- Links Manager and Sub-Daily Report
